Detailed Description

Platform-specific include for the 430BOOST-SHARP96 Memory LCD BoosterPack.

This uses a LS013B4DN04 display. Information on programming it can be found at: http://www.sharpmemorylcd.com/resources.html

#define BSP430_BOOSTERPACK_SHARP96_VCOM   0x40

The command bit pattern that controls VCOM.

This bit may be combined with BSP430_BOOSTERPACK_SHARP96_MODE_DYNAMIC or used in static mode. To eliminate DC bias build-up a command should be sent to the display at a minimum frequency of 1Hz (maximum 30 Hz) while the display is active. This bit should alternate in those commands using a 50% duty cycle.
