BSP430 Coding Style

BSP430 Coding Standard and Style Guide

As BSP430 is intended as a library for use with FreeRTOS, the code style attempts to follow the FreeRTOS Coding Standard and Style Guide. The attempt is best-effort, with several intentional deviations:

  • The C standard type corresponding to the native word size of the MSP430 is int, as it is for many platforms. The type int (unsigned int) will often be used in lieu of portBASE_TYPE. A prefix character i shall be used in lieu of x in function names.

  • While the FreeRTOS variant of Systems Hungarian is used for global type and function names and for parameters in function declarations, it is not used in implementation or in internal structures.

  • Function names comprise the encoding of return type, the identifier BSP430, a lowercase module identifier such as usci, followed by a camel-case initial-capital suffix reflecting the function purpose.

The use of tab characters instead of spaces, with a non-standard spacing of four positions for tables, is retained (regrettably).

5xx/6xx-family devices are identified by checking for defined(MSP430_HAS_MSP430XV2_CPU).

header: periph/%(periph)s.h multiple include protection: BSP430_PERIPH_%(PERIPH)s_H

Global symbols indicate their purpose or flavor using a single prefix character.

struct sBSP430foo is a structure type related to foo. sBSP430foo is a typedef to this struct.

union uBSP430foo is a union type related to foo. uBSP430foo is a typedef to this union.

enum eBSP430foo is an enumeration type related to foo. eBSP430foo is a typedef to this enum.

tBSP430foo is a typedef related to foo that is not a struct, union, or enum. Exception: sBSP430hplPORTfoo is a platform-specific typedef to the variant port structure supported on the target MCU. Exception: hBSP430halPERIPH is a typedef for a pointer to sBSP430halPERIPH.

hBSP430foo is a HAL (hBSP430%(periph)s%(INSTANCE)s) handle for a peripheral.

FreeRTOS HAL state structure: struct sBSP430%(periph)sState FreeRTOS HAL handle type: typedef sBSP430%(periph)sState * tBSP430%(periph)sHandle FreeRTOS HAL handle global: extern tBSP430%(periph)sHandle const hBSP430%(periph)%(INSTANCE)
